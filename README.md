# serde-wincode

[![crates.io](https://img.shields.io/crates/v/serde-wincode.svg)](https://crates.io/crates/serde-wincode)

A `serde` compatibility crate, aiming to replicate `bincode`'s serde compatibility, for `wincode`.
`wincode` + `serde-wincode` should be able to replace `bincode`, [which is now deprecated](https://git.sr.ht/~stygianentity/bincode/commit/a4d858fa026bb87d1dba2edd386e2a6109f827c8).
